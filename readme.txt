== Description ==
Quote your article source at the end of your post

== Installation ==
= Automatic method =
1. Find this plugin in WordPress plugin browser
2. Click install
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In a "Add New" post or page, add up to 3 URLs to quote

or

= Manual method =
1. Download, unzip and upload to your WordPress plugins directory (`/wp-content/plugins/` directory)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In a "Add New" post or page, add up to 3 URLs to quote
